Summary of features exercised
================================================================
- find: default table, --tree --depth, --schema, --columns, --kind,
  --name, --min-size, --sort/--reverse, --format json
- wc:   default totals, --by-kind, --kind, --format json
- cat:  image (accurate VLM caption), video (mosaic + VLM),
  pdf text (-n head), audio transcript
- grep: literal search scoped by --kind
- sql:  --list-tables (pre-index required for ad-hoc queries)
- profile list, config show

Observations
- `mm cat <image>` and `<video>` triggered accurate-mode pipelines by default
  against the active `ollama` profile (gemma4:e2b), not a plain metadata dump.
- `mm sql ... --dir` on an unindexed directory needs `--pre-index`.
- Piping `find | wc` did not propagate file paths via stdin in this run.
