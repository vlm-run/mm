# Video encoder cat snapshots

Recorded outputs for every registered `video-*` encoder, captured against
`~/data/mmbench-tiny/bakery.mp4` (a 4m13s, 28 MB, 1280×720 H.264 clip).

To regenerate:

```bash
mm bench /path/to/bakery-only-dir --command video --format stdout > tests/stdout/cat.md
```

Each block invokes `mm cat <video> --pipeline <encoder>` with `--no-generate`
so output is the raw encoder text (no LLM call). Add `--with-generate` to the
bench command to capture full LLM-rendered output instead.
